# 📂 File Transfer Android to Mac

A simple web-based file sharing application that allows users to transfer files from an Android device to a Mac over the same Wi-Fi network.

---

## 🚀 Features

- 📤 Upload files from Android to Mac
- 💻 Receive files on your Mac instantly
- ⚡ Fast file transfer over the same Wi-Fi network
- 📱 No USB cable required
- 🌐 Easy-to-use web interface
- 📂 Files are automatically saved to the uploads folder

---

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/darshanmr-911/File-Transfer-Android-to-Mac.git
```

### 2. Navigate to the project directory

```bash
cd File-Transfer-Android-to-Mac
```

### 3. Install the required packages

```bash
pip install -r requirements.txt
```

### 4. Run the application

```bash
python app.py
```

---

## 📲 How to Use

1. Connect your Android phone and Mac to the same Wi-Fi network.
2. Start the Flask server.
3. Open the displayed IP address in your Android browser.
4. Choose the file you want to upload.
5. Click **Upload**.
6. The uploaded file will be saved in the **uploads** folder on your Mac.
